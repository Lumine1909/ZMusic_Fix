## Introduction

This is a powerful music system that supports the following functions.

* Full service song
* Play separately
* Lyrics display
* Lyrics translation display
* Multi -search source
* Keyword search
* Personal song list
* Full service song list
* Play of the song list
* Volume adjustment
* Support BungeeCord

## Repositories
